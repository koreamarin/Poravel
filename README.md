## 1️⃣ 개요
- 각 지역의 관광명소 정보 제공 및 정보 공유를 목적으로 관광지 소개 및 이미지, 위치, 리뷰, 추억공유 기능 등을 제공한다.
<br><br>

## 2️⃣ 기술/환경
- 운영체제 : Window 11
- FrontEnd : Vue.js, Vue-router, pinia, JavaScript
- BackEnd : SpringBoot, MyBatis
- DataBase : NCloude DB, MySQL
- API : Kakao Map API
<br><br>

## 3️⃣ 프로젝트 기간
- 2023년 11월 16일 ~ 2023년 11월 23일
